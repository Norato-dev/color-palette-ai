# 🎨 Palette AI

Genera paletas de colores coherentes para UI usando IA. Describe un estilo en texto libre y obtén 5 colores listos para usar en tus proyectos.

## Demo
[palette-ai.vercel.app](https://palette-ai.vercel.app) <!-- actualiza con tu URL real -->

## Features
- 🤖 Generación con IA (Groq + LLaMA 3.3)
- 🎨 Preview en componentes UI reales
- 📋 Exportación en CSS, Tailwind y JSON
- ★ Favoritos con localStorage
- ⚡ Animaciones con Framer Motion

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Groq API (llama-3.3-70b-versatile)
- Framer Motion

## Correr localmente

1. Clona el repo
2. Instala dependencias

```bash
npm install
```

3. Crea tu `.env.local` basado en `.env.example` y agrega tu Groq API Key
4. Corre el proyecto

```bash
npm run dev
```
